Open Source Software

September 5th, 2022


I was excited to learn that we got to learn about open source software. I completed a web development bootcamp a few months ago and my instructors encouraged me to contribute to open source projects in order to better practice my web development skills. I have been nervous to do so and had not taken the time to learn about open source until I read the assigned readings for my IP&T 521 class. The readings offered more clarity on what open source software actually is. The reading from opensource.org was specifically useful when it indicated that open source software must allow for free redistrubution, provide the source code, and allow for modified works (The Open Source Definition).

I did not recognize most of the open source software that was mentioned in the different articles. Linux was the resource that I recognized and I know that it has had a large impact in the world of computer science. I was curious what open-source softwares exist today and with the help of google I realized that React, a web development library, is actually considered open-source software. The frontend web library, React, was actually developed by the company Facebook and is what is used to develop Facebook, Instragram and many other online applications. The software was then released as an open-source resource in May of 2013 (Wikipedia). React is now used by many companies and many different people outside of Meta can contribute to its growth. This seems to be an open-source success story.

Spending time learning about open-source resources has relit my desire to contribute to open source projects. My web development skills are pretty shabby, but I found a useful blog, called First Timers Only, that helps beginner web developers learn how to successfully contribute to open-source projects. I will hopefully attempt to make some small contribution to a project in the future.
